# vehicle_dynamics_report

A full course work report for a vehicle dynamics master class done using Matlab. The following are topics covered in the report:
* Tire Modelling
  * Longitudnial and lateral slip
  * Pacejka magic formula
* Sensor data measurements from IMU
* Double and single track model
  * Steering and handling behaviour
  * Handling Diagrams
* Planning and Control
  * Route planning, High level planning/ control, Low level control
  * Model predictive control (MPC)
  * Lateral motion controls
    * Pure pursuit controller
    * kinematic, dynamic stanley controller
    * Clothoid-based controller
  * Sampling based methods: RRT and RRT*
  * Exploration methods: A*
